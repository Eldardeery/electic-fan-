# Multi-Functional-Electric-Fan
##A multi-functional electric fan with    
 1. an auto-off timer,   
 2. increase and decrease the speed    
 3. change the direction    
 4. a 4 7-segments display to show the time    
 5. lcd to show the speed and direction
